Securing Deep Learning Models on NLP

Synonym Substitution Attack

This bert-base-uncased model was fine-tuned for comment classification and the yelp_polarity dataset loaded using the nlp library. The model was fine-tuned for 10 epochs with a batch size of 16, a learning rate of 5e-05, and a maximum sequence length of 256. Since this was a classification task, the model was trained with a cross-entropy loss function. The best score the model achieved on this task was 0.99, as measured by the eval set accuracy, found after 4 epochs.


Adversarial Examples which can fool the model

This model can be fooled by the adversarial examples generated against this model using the text attack library, Text Fooler as a recipe and in order to combat those attacks a Normalised Model has been developed with embedding surgery. which can greatly improves the robustness of the attack to fine-tuning.
